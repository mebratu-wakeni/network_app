# Stock CSV import samples

Use with **Inventory → Stock → Import**.

Expiry dates must be **dd/mm/yyyy** when present.

| File | Purpose |
|------|---------|
| `stock-import-valid.csv` | Happy-path stock rows |
| `stock-import-invalid.csv` | Invalid rows for error handling |

See also [`../products-import/`](../products-import/) for product master import.
